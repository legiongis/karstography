Sink Locations (Crawford Co.)
--------------

Database exports to shapefile, 2024-09-30. Sink locations `derived from LiDAR <_static/files/Sinkhole_Methodology_Report.pdf>`_, categorized `by our volunteers <evaluating-sinks.html>`_.

- `all sinks <https://csp.us-southeast-1.linodeobjects.com/exports/sinks20240930.zip>`_
    - 10,636 features, all "sink_type" categories
- `sinkholes <https://csp.us-southeast-1.linodeobjects.com/exports/sinks20240930_SINKHOLE.zip>`_
    - 455 features, only sinks categorized as sinkholes
    - ``confidence`` = "POSSIBLE" or "PROBABLE" 
- `catchment basins <https://csp.us-southeast-1.linodeobjects.com/export/sinks20240930_CATCHMENT.zip>`_
    - 165 features, only sinks categorized as catchment basins ("cattle ponds", etc.)
